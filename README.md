# McDonald's Sales Data Analysis using Excel

## Project Overview
This project analyzes McDonald's sales data using Microsoft Excel to identify revenue trends, product performance, and customer ordering behavior. The analysis is presented through an interactive Excel dashboard.

## Dataset
The dataset contains McDonald's menu items including item name, category, and price. It is used to analyze product performance and revenue contribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization
- Excel Dashboard

## Dashboard
The interactive Excel dashboard provides insights into:
- Category-wise revenue
- Top selling menu items
- Monthly sales trends
- Customer ordering behavior

## Key Insights
- Burger category generated the highest revenue.
- Afternoon time slot recorded the highest order volume.
- Weekday sales were higher compared to weekends.
- Top 5 menu items contributed significantly to total revenue.

## Business Recommendations
- Focus promotional campaigns on high-performing categories.
- Introduce offers during low sales time slots.
- Bundle top-selling items to increase average order value.
- Strengthen weekend marketing strategies.
